## What is this?

**anonymouswithAtok** is Nairobi's raw, unfiltered anonymous confession board. Users write freely — no accounts, no IP logging, no records. Confessions go straight to Instagram DM at [@notjustatok](https://instagram.com/notjustatok).

Built for Kenya. Designed for the streets.

## Features

- 🔥 Anonymous confession submission — no character limit
- 📖 Magazine-style blog feed with likes, shares, and read counts
- 📲 Direct to Instagram DM — no backend needed
- 🎨 Nairobi Noir aesthetic — dark, cinematic, red
- ✨ Ember particle canvas, cursor glow, scanline effects
- 🔒 Privacy blur while typing (shoulder-surf protection)
- 📱 Mobile-first — built for Kenyan smartphone users
- 🌍 Full SEO — Open Graph, Twitter Cards, JSON-LD, Sitemap

---

## Tech Stack

- **React 18** + **Vite**
- **Azeret Mono** + **Instrument Serif** (Google Fonts)
- **Vercel** deployment
- **Canvas API** for particle effects
- Zero backend — 100% frontend

---

## Deploy

```bash
npm install
npm run dev       # local development
npm run build     # production build
```

Deploy to Vercel → connect GitHub repo → auto-deploys on every push.
